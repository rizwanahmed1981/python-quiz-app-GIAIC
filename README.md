## 🚀 About the Project  

**Random Quiz App** is an interactive web application that generates random quiz questions on various topics. Built with **Python, Streamlit, and uv**, it provides a smooth and engaging user experience for quiz lovers.  

🔗 **Live Demo**: [Check it out on Streamlit](https://python-quiz-app-giaic.streamlit.app/)  
📂 **GitHub Repository**: [GitHub Link](https://github.com/rizwanahmed1981/python-quiz-app-GIAIC)  

---

## ✨ Features  

✅ Generates **random quiz questions** dynamically  
✅ **Interactive** and **user-friendly** UI  
✅ **Instant feedback** on answers  
✅ **Lightweight & fast performance** with uv  
✅ **Easy to deploy and extend**

## 📥 Installation  

To run this project locally, follow these steps:  

### 1️⃣ Clone the Repository  
```bash
git clone https://github.com/rizwanahmed1981/python-quiz-app-GIAIC.git
cd python-quiz-app-GIAIC
```

### 2️⃣ Create a Virtual Environment (Optional but Recommended)  
```bash
python -m venv venv
source venv/bin/activate  # On macOS/Linux
.venv\Scripts\activate     # On Windows
```

### 3️⃣ Install Dependencies  
```bash
uv add -r requirements.txt
```

### 4️⃣ Run the App  
```bash
streamlit run main.py
```
